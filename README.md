<h1 align="center">🤖 Multi-Agent AI Research System</h1>

<p align="center">
An intelligent multi-agent AI platform designed to automate web research, information extraction, reasoning, and knowledge synthesis using collaborative AI agents.
</p>

<p align="center">
Built with LangChain, Mistral AI, React + Vite, Tavily API, and BeautifulSoup.
</p>

<h2>🧠 System Architecture</h2>

<p>
The platform follows a collaborative multi-agent workflow where specialized AI agents work together to complete complex research tasks efficiently.
</p>

<table>
  <tr>
    <th>Agent</th>
    <th>Responsibility</th>
  </tr>
  <tr>
    <td>Research Agent</td>
    <td>Performs intelligent web searches</td>
  </tr>
  <tr>
    <td>Scraper Agent</td>
    <td>Extracts relevant web content</td>
  </tr>
  <tr>
    <td>Analysis Agent</td>
    <td>Processes and evaluates gathered information</td>
  </tr>
  <tr>
    <td>Summarization Agent</td>
    <td>Generates concise research outputs</td>
  </tr>
  <tr>
    <td>Coordinator Agent</td>
    <td>Manages workflow and agent communication</td>
  </tr>
</table>

<hr>

<h2>⚙️ How It Works</h2>

<ol>
  <li>User submits a research query</li>
  <li>Research agents perform live web searches</li>
  <li>Scraper agents extract relevant information</li>
  <li>AI agents analyze and validate the data</li>
  <li>Summarization agents synthesize final insights</li>
  <li>Results are displayed through the frontend</li>
</ol>
